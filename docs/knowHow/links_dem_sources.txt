-> Zip von https://sonny.4lima.de/ herunterladen
-> hgt file extrahieren
-> in F:\Ortho4XP\Elevation_data entsprechend platzieren
-> in Ortho4xp die Schritte "Assemble Vector Data" und "Triangulate Mesh" wiederholen lassen (einzeln oder Batch fÃ¼r mehrere Files)
-> App Config:
	- custom dem Pfad bleibt leer!
	- Base Folder muss der sein, wo die aktuellen Tiles liegen.
